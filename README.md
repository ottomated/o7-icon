<p align="center">The most efficient Svelte icon library,</p>

## Basic Usage

```svelte
<script>
	import { UserPlus } from '@o7/icon/lucide';
</script>

<UserPlus />
```

When you first use an icon, it includes the full svg. Any subsequent uses only need this:
